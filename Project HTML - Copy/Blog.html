Welcome to the behind-the-scenes of my work.

Bluetooth-Controlled Lamp

I built a Bluetooth-controlled lamp using an Arduino UNO, a Bluetooth module (HC-05), a relay module, and a simple lamp. I connected the lamp to the relay so that it can be switched on and off using my phone via Bluetooth.

The project taught me how to integrate hardware with software, how to use serial communication, and how to safely work with electronics.

My Portfolio Website

I designed and coded my personal portfolio website using HTML and CSS. It includes sections like Home, About, Services, Blog, Testimonials, and Resume.

The goal was to showcase my skills and allow people to get in touch with me. I also learned a lot about layout, styling, and responsive design.
